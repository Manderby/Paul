Quick Start:
------------

- Download whole package.
  git clone --recurse-submodules https://github.com/Manderby/Plotty.git
  
- Use CMake to build the Plotty project in the code folder.



Description:
------------

Plotty is a small demo function plotter application to show the capabilities of NALib.

Email: plotty@manderc.com
