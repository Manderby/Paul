Quick Start:
------------

- Download whole package.
  git clone --recurse-submodules https://github.com/Manderby/Paul.git
  
- Use CMake to build the Paul project in the code folder.



Description:
------------

Paul is a small demo function plotter application to show the capabilities of NALib.
